## Qué es

A11Y Github Actions es un GitHub Action open source.

## Para qué sirve

- Automatizar pasos en tus workflows de CI/CD con A11Y Github Actions.
- Reducir configuración manual en GitHub Actions.
- Encadenar esta acción con otras del marketplace de GitHub.
- Aumentar la fiabilidad con tests automatizados.

## Cuándo usarlo

- Cuando usas GitHub Actions y quieres automatizar un paso en tus workflows sin escribirlo desde cero.

## Puntos clave

- Se integra con el marketplace de GitHub Actions.
- Configurable mediante inputs en el workflow YAML.
- Open source y disponible en GitHub.

## Ten en cuenta

- Configura correctamente los permisos y tokens de seguridad del workflow.
- Consulta la documentación oficial para obtener la información más actualizada.
