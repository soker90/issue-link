## Qué es

JSON Data AI es una herramienta para generar datos JSON usando inteligencia artificial. En vez de escribir fixtures manualmente, describes el tipo de datos que necesitas y obtienes una estructura lista para copiar o adaptar.

## Para qué sirve

- Crear datos de ejemplo para prototipos.
- Generar fixtures para tests.
- Preparar mocks para demos o documentacion.
- Rellenar interfaces mientras el backend real aun no existe.

## Cuándo usarlo

Usalo cuando necesitas datos realistas rapido: usuarios, productos, ordenes, comentarios, configuraciones, eventos o cualquier estructura repetitiva. Es util en frontend, pruebas de concepto y documentacion de APIs.

## Puntos clave

- Ahorra tiempo escribiendo JSON repetitivo.
- Permite crear datasets con contexto semantico.
- Encaja muy bien en prototipado y tests manuales.

## Ten en cuenta

Revisa siempre el resultado antes de usarlo en tests importantes. La IA puede inventar campos inconsistentes o generar estructuras que no cumplen exactamente tu contrato.
