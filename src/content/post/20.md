## Qué es

Squoosh es una aplicacion web para comprimir, convertir y comparar imagenes. Permite ajustar calidad, formato y parametros de compresion viendo el resultado antes de descargarlo.

## Para qué sirve

- Reducir el peso de imagenes antes de subirlas a una web.
- Convertir entre formatos como JPEG, PNG, WebP o AVIF.
- Comparar visualmente calidad y tamano final.
- Preparar assets optimizados para performance web.

## Cuándo usarlo

Usalo cuando tienes pocas imagenes y quieres optimizarlas manualmente con control visual. Es ideal para imagenes de landing, screenshots, banners o assets estaticos.

## Puntos clave

- Funciona desde el navegador.
- Da control fino sobre formato y calidad.
- Muy util para aprender el impacto real de cada codec.

## Ten en cuenta

Para pipelines grandes o muchas imagenes, es mejor automatizar con herramientas de build, scripts o servicios de optimizacion.
