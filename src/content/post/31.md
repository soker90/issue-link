## Qué es

Upptime es una solucion open source para monitorizar uptime y publicar una pagina de estado usando GitHub Actions, GitHub Issues y GitHub Pages. Todo el sistema vive en GitHub.

## Para qué sirve

- Monitorizar webs, APIs o endpoints HTTP.
- Publicar automaticamente una status page.
- Registrar incidentes como issues de GitHub.
- Tener historico de disponibilidad sin contratar infraestructura adicional.

## Cuándo usarlo

Usalo si quieres una monitorizacion transparente, gratuita y versionada en GitHub. Es ideal para proyectos personales, servicios pequenos, APIs publicas o herramientas internas donde GitHub ya forma parte del flujo de trabajo.

## Puntos clave

- Funciona con GitHub Actions.
- Publica estado en GitHub Pages.
- Usa issues para incidentes.
- No necesita servidor propio.

## Ten en cuenta

Depende de la ejecucion programada de GitHub Actions, por lo que no es lo mismo que un monitor comercial con checks distribuidos globalmente y alertas avanzadas. Para muchos proyectos pequenos, esa simplicidad es precisamente la ventaja.
