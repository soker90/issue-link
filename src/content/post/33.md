## Qué es

Modern Web Guidance es un proyecto del equipo de Google Chrome que empaqueta conocimiento experto sobre la plataforma web (APIs modernas del navegador, CSS, formularios, rendimiento, accesibilidad y datos de compatibilidad Baseline) en forma de "skills" para agentes de codificación con IA. Este repositorio, `modern-web-guidance-src`, contiene el código fuente con el que se generan los más de 100 casos de uso verificados; el paquete publicado se distribuye aparte como `modern-web-guidance` en npm. Se publica bajo licencia Apache 2.0 (la documentación bajo CC BY 4.0).

## Para qué sirve

- Evitar que los agentes de IA recurran a patrones obsoletos por culpa de datos de entrenamiento antiguos.
- Inyectar en el contexto del agente guías concretas con código, gotchas y fallbacks responsables en vez de polyfills pesados.
- Dar a los agentes datos de compatibilidad de navegadores basados en Baseline al sugerir soluciones.
- Integrar buenas prácticas de la web moderna en flujos de trabajo con Claude Code, Antigravity, GitHub Copilot CLI, Gemini CLI, Goose o Vercel Agent Skills.

## Cuándo usarlo

Es útil cuando trabajas con un agente de codificación y quieres que sus sugerencias para frontend sigan estándares actuales en lugar de soluciones de hace varios años. El agente activa la skill ante una tarea relevante y consulta el CLI; la búsqueda usa un modelo TensorFlow.js offline y eficiente en CPU, sin llamadas de red ni claves de API. Si solo necesitas documentación de referencia para humanos, MDN o el propio Chrome for Developers cumplen ese papel; esto está pensado específicamente para alimentar agentes.

## Ejemplo

Instalación interactiva del paquete publicado:

```bash
npx modern-web-guidance@latest install
```

Una vez instalado, el agente consulta y recupera guías por su cuenta:

```bash
modern-web-guidance search "responsive image layout"
modern-web-guidance retrieve <guide-id>
```

## Puntos clave

- Búsqueda semántica local con TensorFlow.js: sin red, sin claves de API y eficiente en CPU.
- Paquete autocontenido y sin dependencias externas, pensado para baja latencia y seguridad de la cadena de suministro.
- Contenido optimizado en tokens mediante bucles de evaluación automatizados, con mejoras medibles en la calidad del código generado por el agente.

## Ten en cuenta

Es una herramienta orientada a agentes de codificación, no una librería que integres directamente en tu aplicación; su valor depende de usar un agente compatible. Este repositorio es la fuente (`-src`) con la que se construye el contenido: para usarlo en el día a día instala el paquete `modern-web-guidance` desde npm. El proyecto está en desarrollo activo y su versionado aún es temprano, por lo que la API y los comandos pueden cambiar.
