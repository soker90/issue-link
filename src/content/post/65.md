## Qué es

OpenStove es una plataforma comunitaria para descubrir y compartir recetas de cocina, creada por mearashadowfax. Está construida con Astro como sitio estático y se apoya en la biblioteca de componentes Preline. Su propuesta es ofrecer una base de recetas curada por aficionados de todo el mundo, libre de anuncios y de distracciones.

## Para qué sirve

- Buscar y explorar recetas en una web rápida y sin publicidad.
- Aportar tus propias recetas a la comunidad mediante pull requests.
- Estudiar un proyecto Astro real como referencia para tu propio sitio de contenido.
- Hacer fork del proyecto para montar tu propia colección de recetas.

## Cuándo usarlo

- Cuando quieres un recetario abierto y colaborativo en lugar de webs llenas de anuncios y rastreadores.
- Cuando buscas un ejemplo práctico de sitio de contenido hecho con Astro y contribución vía Git.
- Si prefieres alternativas open source frente a plataformas de recetas cerradas y comerciales.

## Ejemplo

El proyecto se desarrolla como cualquier sitio Astro. Para levantarlo en local tras clonar el repositorio:

```
git clone https://github.com/mearashadowfax/OpenStove.git
cd OpenStove
npm install
npm run dev
```

Para contribuir con una receta, se sigue el formato indicado en CONTRIBUTING.md y se abre un pull request con la receta para que la comunidad la disfrute.

## Puntos clave

- Plataforma de recetas totalmente gratuita y sin anuncios.
- Contribución abierta: las recetas se añaden mediante pull requests en GitHub.
- Construida con Astro y componentes Preline, con web oficial en openstove.org.

Es un proyecto de comunidad relativamente pequeño, así que el volumen de recetas depende de las contribuciones que vaya recibiendo.
