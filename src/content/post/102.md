## Qué es

Bigspring Light Astro es una plantilla (theme) gratuita y de código abierto creada por Themefisher, pensada específicamente para sitios web de SaaS y marketing. Está construida con Astro y Tailwind CSS, e incluye más de 10 páginas prediseñadas (inicio, blog, precios, contacto, FAQ…) listas para personalizar. Se distribuye con licencia MIT.

## Para qué sirve

- Lanzar rápidamente una landing o web corporativa de un producto SaaS sin partir de un lienzo en blanco.
- Reaprovechar páginas ya maquetadas (precios, blog, FAQ, contacto) y centrarte solo en el contenido.
- Editar el contenido en Markdown/MDX, con componentes autoimportados en MDX.
- Desplegar fácil en Netlify o Vercel (trae configuración lista) con buen rendimiento (PageSpeed 95+).

## Cuándo usarlo

Encaja cuando necesitas una web de marketing o SaaS rápida, responsive y con buen SEO, y prefieres Astro + Tailwind. Es buena base si quieres algo gratuito y editable en lugar de un constructor visual de pago. Si necesitas funcionalidades muy a medida o un panel de administración complejo, te servirá más como punto de partida que como solución final.

## Ejemplo

Clonar la plantilla y arrancarla en local:

```bash
# Clonar el tema
git clone https://github.com/themefisher/bigspring-light-astro.git
cd bigspring-light-astro

# Instalar dependencias y arrancar en desarrollo
npm install
npm run dev

# Generar la build de producción
npm run build
```

## Puntos clave

- Más de 10 páginas prediseñadas y diseño totalmente responsive.
- Construida con Astro y Tailwind CSS; contenido en Markdown/MDX.
- SEO optimizado con soporte de imágenes Open Graph y formulario de contacto.
- Configuración lista para desplegar en Netlify e integración con Sitepins CMS para edición visual.
- Open source con licencia MIT y mantenida activamente.

## Ten en cuenta

- Es un punto de partida: tendrás que adaptar textos, marca y secciones a tu producto.
- Orientada a webs de marketing/SaaS; para una app con mucha lógica de cliente o backend propio se queda corta por sí sola.
- Themefisher ofrece versiones "pro" de pago de sus temas con más páginas y soporte; esta es la variante gratuita ("light").
