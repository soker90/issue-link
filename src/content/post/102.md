## Qué es

Bigspring Light Astro es una plantilla (theme) gratuita y de código abierto creada por Themefisher, orientada a sitios de SaaS y marketing. Está construida con Astro y Tailwind CSS, con contenido editable en Markdown/MDX y soporte de componentes autoimportados. Incluye más de 10 páginas prediseñadas (inicio, blog, precios, contacto, FAQ, elementos…) listas para personalizar y se distribuye con licencia MIT.

## Para qué sirve

- Lanzar rápido una landing o web corporativa de un producto SaaS sin partir de un lienzo en blanco.
- Reaprovechar páginas ya maquetadas (precios, blog, FAQ, contacto) y centrarte solo en el contenido.
- Personalizar colores, tipografías, menú, enlaces sociales y metaetiquetas SEO desde la configuración.
- Desplegar con buen rendimiento (PageSpeed 95+) y configuración lista para Netlify.

## Cuándo usarlo

Encaja cuando necesitas una web de marketing o SaaS rápida, responsive y con buen SEO, y prefieres trabajar con Astro y Tailwind. Es buena base si quieres algo gratuito y editable en Markdown en lugar de un constructor visual de pago. Si necesitas funcionalidades muy a medida o un panel de administración complejo, te servirá más como punto de partida que como solución final. Themefisher ofrece variantes "pro" de pago con más páginas y soporte.

## Ejemplo

Clonar la plantilla y arrancarla en local:

```bash
# Clonar el tema
git clone https://github.com/themefisher/bigspring-light-astro.git
cd bigspring-light-astro

# Instalar dependencias y arrancar en desarrollo
npm install
npm run dev

# Generar la build de producción
npm run build
```

## Puntos clave

- Más de 10 páginas prediseñadas y diseño totalmente responsive.
- Construida con Astro y Tailwind CSS; contenido en Markdown/MDX con componentes autoimportados.
- Altamente personalizable: color, fuente, menú, enlaces sociales y metaetiquetas SEO.
- Optimizada para rendimiento (95+ en Google PageSpeed) y con configuración lista para Netlify.

## Ten en cuenta

- Es un punto de partida: tendrás que adaptar textos, marca y secciones a tu producto.
- Orientada a webs de marketing/SaaS; para una app con mucha lógica de cliente o backend propio se queda corta por sí sola.
- La variante gratuita es "light"; Themefisher vende versiones "pro" con más páginas y soporte de pago.
