## Qué es

UIverse es una galeria comunitaria de componentes visuales hechos principalmente con HTML, CSS y Tailwind. Incluye botones, loaders, inputs, toggles, cards, animaciones y pequenas piezas de interfaz.

## Para qué sirve

- Encontrar inspiracion para detalles de UI.
- Copiar componentes pequenos y adaptarlos.
- Descubrir microinteracciones para mejorar una interfaz.
- Ahorrar tiempo en elementos visuales no criticos.

## Cuándo usarlo

Usalo si necesitas un boton llamativo, un loader, una card animada o una interaccion concreta y no quieres disenarla desde cero. Funciona muy bien como repositorio de ideas visuales.

## Puntos clave

- Muchisima variedad de componentes pequenos.
- Util para inspiracion rapida.
- Muchos ejemplos son faciles de adaptar.

## Ten en cuenta

La calidad varia porque es contenido comunitario. Revisa accesibilidad, responsive, contraste y complejidad CSS antes de usar algo en produccion.
