## Qué es

Shots es una herramienta visual para crear mockups y composiciones atractivas a partir de capturas de pantalla. Permite transformar una captura normal en una imagen mas presentable para marketing, portfolios, redes sociales o documentacion.

## Para qué sirve

- Crear mockups de una app o web sin abrir una herramienta de diseno pesada.
- Preparar imagenes para README, landing pages o posts.
- Presentar capturas con fondos, marcos y estilos mas cuidados.
- Generar material visual rapido para mostrar avances de producto.

## Cuándo usarlo

Usalo cuando tienes una captura funcional pero necesitas que parezca mas profesional. Es util para proyectos personales, anuncios de features, demos y portfolios.

## Puntos clave

- Muy rapido para pasar de captura simple a imagen presentable.
- Buen recurso para contenido visual de producto.
- No sustituye a Figma o herramientas de diseno completas, pero reduce mucho la friccion.

## Ten en cuenta

Si necesitas composiciones complejas, sistema de componentes o edicion precisa, probablemente quieras usar una herramienta de diseno dedicada.
