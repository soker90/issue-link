## Qué es

Resend es un servicio para enviar emails transaccionales desde aplicaciones. Esta orientado a desarrolladores y ofrece una API sencilla para mandar correos desde backend, serverless functions o frameworks modernos.

## Para qué sirve

- Enviar emails de registro, verificacion o recuperacion de password.
- Mandar magic links o codigos de acceso.
- Enviar notificaciones transaccionales.
- Gestionar emails de formularios, invitaciones o avisos de producto.

## Cuándo usarlo

Usalo si necesitas enviar emails desde una app y no quieres montar ni mantener infraestructura SMTP. Es especialmente comodo en proyectos con Next.js, Remix, Astro SSR, APIs Node o funciones serverless.

## Puntos clave

- API clara y SDK para JavaScript/TypeScript.
- Buen encaje con emails construidos en React Email.
- Tiene plan gratuito suficiente para prototipos y proyectos pequenos.

## Ejemplo basico

```javascript
import { Resend } from 'resend';

const resend = new Resend(process.env.RESEND_API_KEY);

await resend.emails.send({
  from: 'app@example.com',
  to: 'user@example.com',
  subject: 'Bienvenido',
  html: '<p>Tu cuenta esta lista.</p>',
});
```

## Ten en cuenta

Como cualquier servicio de email, requiere configurar dominios, DNS, SPF/DKIM y buenas practicas de entregabilidad si vas a usarlo en produccion.
