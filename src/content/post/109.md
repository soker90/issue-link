## Qué es

Link Maker es una pequeña herramienta web open source creada por Kuizuo que convierte una URL en una tarjeta de previsualización con estilo. Está construida con Fresh, el framework de Deno, en TypeScript y con Twind para los estilos, y se publica bajo licencia MIT. Tiene una demo en línea funcionando.

## Para qué sirve

- Generar una tarjeta visual de un enlace a partir de su URL, con su título, descripción e imagen.
- Obtener una previsualización lista para incrustar o compartir sin tener que maquetarla a mano.
- Servir de ejemplo práctico de una app construida con Deno y Fresh.
- Hacer fork y adaptarla como base para tu propio generador de previews.

## Cuándo usarlo

Es útil cuando necesitas mostrar enlaces como tarjetas atractivas (en un blog, una newsletter o una página de recursos) y quieres una solución sencilla y gratuita. Para necesidades más completas o con API existen servicios dedicados como LinkPreview o Microlink; Link Maker es una alternativa ligera y autoalojable.

## Ejemplo

Flujo de uso de la demo en línea:

1. Abre la app de Link Maker en el navegador.
2. Pega en el campo de entrada la URL que quieres previsualizar.
3. La herramienta lee los metadatos de la página y genera una tarjeta con su título, descripción e imagen.
4. Copia o reutiliza la tarjeta resultante.

Para ejecutarla en local con Deno:

deno task start

## Puntos clave

- Convierte cualquier URL en una tarjeta de previsualización con estilo card.
- Construida sobre Deno y Fresh, con renderizado en el servidor.
- Proyecto open source con licencia MIT y demo pública disponible.

## Ten en cuenta

Es un proyecto pequeño y de propósito acotado, con poca actividad reciente en el repositorio, así que conviene revisarlo antes de depender de él en producción. Requiere Deno para ejecutarlo en local. Si necesitas extracción de metadatos a escala o vía API, valora servicios especializados como Microlink o LinkPreview.
