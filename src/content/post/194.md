## Qué es

Mantis Free React Admin Template es una plantilla gratuita y de código abierto para construir paneles de administración (dashboards) con React. La desarrolla y mantiene CodedThemes, y combina el sistema de componentes de Material UI (MUI) con los principios de diseño de Ant Design. Se distribuye bajo licencia MIT e incluye dos variantes en el mismo repositorio: una versión SPA con Vite (la principal) y otra basada en Next.js.

## Para qué sirve

- Montar el esqueleto de un panel de administración con navegación lateral, cabecera y rutas ya cableadas.
- Reutilizar componentes de interfaz ya estilizados (tarjetas, tablas, formularios, gráficos) sin diseñarlos desde cero.
- Disponer de tema claro y oscuro y de un sistema de personalización basado en Material UI.
- Aprovechar una estructura de proyecto organizada como base para una aplicación interna o un producto.
- Servir como punto de partida antes de plantearse la versión de pago (Mantis Pro) con más páginas y componentes.

## Cuándo usarlo

Es buena opción cuando quieres un dashboard en React con la estética de Material UI y necesitas avanzar rápido sin construir el layout y los componentes base. Si tu proyecto usa Next.js, el repositorio incluye una variante específica; si es una SPA clásica, usa la de Vite. Como alternativas gratuitas similares existen Materio, Berry o Minimal UI, también basadas en MUI.

## Ejemplo

Clona el repositorio, entra en la variante de Vite, instala dependencias y arranca el servidor de desarrollo:

```bash
git clone https://github.com/codedthemes/mantis-free-react-admin-template.git
cd mantis-free-react-admin-template/vite

# Instalar dependencias (usa yarn o npm)
yarn

# Arrancar en modo desarrollo
yarn start

# Generar la build de producción
yarn build
```

Para la versión con Next.js, entra en la carpeta `next` en lugar de `vite` y ejecuta los mismos comandos.

## Puntos clave

- Construida con React 19, Material UI v9 y React Router 7 (versión Vite, v2.2.0).
- Mezcla los componentes de MUI con criterios de diseño tomados de Ant Design.
- Incluye soporte de tema claro y oscuro y diseño totalmente responsive.
- Ofrece dos variantes en el mismo repositorio: SPA con Vite y aplicación con Next.js.
- Código abierto bajo licencia MIT, con documentación oficial en GitBook.

## Ten en cuenta

- La versión gratuita incluye un número limitado de páginas y componentes de demostración; el catálogo completo (más de 100 páginas, autenticación, multi-idioma) está en la versión de pago Mantis Pro.
- Está pensada para el ecosistema Material UI: si tu proyecto usa otro sistema de componentes, encajará peor.
- Como toda plantilla, conviene personalizarla a fondo para que tu aplicación no quede genérica.
