## Qué es

Tauri es un framework para crear aplicaciones de escritorio usando una interfaz web y una capa nativa escrita en Rust. Permite empaquetar apps multiplataforma con binarios mas pequenos que muchas alternativas basadas en Chromium completo.

## Para qué sirve

- Convertir una app web en una app de escritorio.
- Crear herramientas internas multiplataforma.
- Acceder a capacidades nativas desde una UI HTML/CSS/JS.
- Reducir tamano y consumo frente a soluciones mas pesadas.

## Cuándo usarlo

Usalo si ya dominas frontend web y quieres distribuir una app desktop para Windows, macOS o Linux. Es especialmente interesante para herramientas de desarrollo, clientes locales, dashboards offline o utilidades que necesitan acceso al sistema.

## Puntos clave

- Usa tecnologias web para la interfaz.
- La parte nativa se apoya en Rust.
- Binarios normalmente mas pequenos que Electron.
- Buen enfoque de seguridad por permisos y APIs controladas.

## Ten en cuenta

No es simplemente una web empaquetada. Para aprovecharlo bien conviene entender su modelo de comandos, permisos, bundling y comunicacion entre frontend y backend nativo.
