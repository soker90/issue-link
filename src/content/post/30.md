## Qué es

CSS Code Quality Analyzer es una herramienta de Project Wallace para analizar una hoja de estilos y detectar indicadores de complejidad, mantenibilidad y posibles problemas de rendimiento.

## Para qué sirve

- Auditar CSS existente.
- Detectar selectores demasiado complejos.
- Revisar repeticion, especificidad y tamano.
- Identificar deuda tecnica antes de que el CSS sea inmanejable.

## Cuándo usarlo

Usalo al heredar una web, antes de refactorizar estilos o cuando notas que cambiar CSS empieza a romper partes inesperadas. Tambien sirve como control de calidad antes de lanzar una version importante.

## Puntos clave

- Da una vision objetiva del estado del CSS.
- Ayuda a justificar refactors con datos.
- Puede revelar problemas que no se ven mirando solo la UI.

## Ten en cuenta

No sustituye una revision manual. Una metrica puede avisar de complejidad, pero necesitas contexto para decidir si una parte del CSS realmente merece ser refactorizada.
