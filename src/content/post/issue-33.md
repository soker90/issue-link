jsDelivr es una red de entrega de contenido (CDN) pública y gratuita para proyectos de código abierto. Puede servir archivos directamente desde el registro npm y repositorios de GitHub sin configuración previa, ofreciendo un rendimiento rápido y confiable a nivel mundial.
