## Qué es

Supabase es la alternativa open source a Firebase más completa. Construida sobre Postgres, ofrece una base de datos relacional real con todas sus ventajas: queries SQL, joins, RLS (Row Level Security), funciones y extensiones como pgvector para embeddings.

## Para qué sirve

- Tener autenticación completa (email, OAuth con Google/GitHub/etc.) sin escribir una línea de backend.
- Exponer la base de datos como API REST y GraphQL de forma automática.
- Almacenar y servir archivos con un servicio de storage compatible con S3.
- Recibir cambios en tiempo real de la base de datos mediante websockets.
- Ejecutar lógica de servidor con Edge Functions (Deno).

## Cuándo usarlo

Es una opción excelente para proyectos SaaS, apps móviles y dashboards donde quieres un backend funcional rápido sin gestionar servidores. También funciona muy bien como capa de datos para proyectos con Next.js, Astro o SvelteKit.

## Puntos clave

- Postgres real, no una abstracción: puedes usar cualquier cliente SQL o herramienta que soporte Postgres.
- Row Level Security permite definir qué usuario puede ver qué datos directamente en la base de datos.
- SDK oficial para JavaScript/TypeScript, Python, Dart y más.
- Tier gratuito generoso para proyectos pequeños y experimentos.

## Ten en cuenta

El tier gratuito pausa los proyectos inactivos después de una semana. En producción conviene planificar el tier de pago. Si necesitas control total sobre la infraestructura, se puede desplegar en autohosting.
