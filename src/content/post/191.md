## Qué es

Million React es un proyecto starter (plantilla) de código abierto creado por Aiden Bai, el autor de Million.js. Consiste en un esqueleto mínimo basado en Vite y React con Million.js ya integrado, pensado para experimentar con las optimizaciones de renderizado que aporta la librería. El repositorio está archivado desde septiembre de 2023 y es de solo lectura.

## Para qué sirve

- Arrancar un proyecto React + Vite con Million.js ya configurado, sin partir de cero.
- Probar rápidamente el "block virtual DOM" de Million.js sobre componentes reales.
- Servir de referencia mínima de cómo enganchar Million.js en una app de Vite.
- Evitar la configuración inicial repetitiva al evaluar la librería.

## Cuándo usarlo

Es útil solo como punto de partida para hacer pruebas con Million.js en un entorno Vite. Al estar archivado, conviene tratarlo como ejemplo histórico: para un proyecto real es mejor crear un proyecto Vite actual y seguir la guía de instalación oficial de Million.js, que ofrece integraciones mantenidas para Vite, Next.js y Astro.

## Ejemplo

Clonar el starter y levantar el entorno de desarrollo:

```bash
git clone https://github.com/aidenybai/million-react.git
cd million-react
npm install
npm run dev
```

Sobre un proyecto Vite propio, la forma actual de añadir Million.js es mediante su plugin de compilación en `vite.config.js`:

```js
import million from "million/compiler";
import react from "@vitejs/plugin-react";
import { defineConfig } from "vite";

export default defineConfig({
  plugins: [million.vite({ auto: true }), react()],
});
```

## Puntos clave

- Plantilla lista para producción con la estructura básica de Vite + React.
- Million.js convierte la reconciliación de React de O(n) a O(1) actualizando directamente los nodos del DOM afectados.
- Mantenido por el mismo autor que la librería Million.js.
- Código abierto y disponible en GitHub.

## Ten en cuenta

El repositorio está **archivado** (solo lectura) desde 2023, por lo que sus dependencias y la versión de Million.js que incluye están desfasadas. La propia librería Million.js (repo `aidenybai/million`) sigue siendo la referencia y conviene instalarla con su versión y plugin actuales en lugar de depender de este starter. Además, el equipo detrás de million.dev ha reorientado su actividad hacia herramientas de IA, así que verifica el estado de mantenimiento de Million.js antes de adoptarlo en un proyecto nuevo.
