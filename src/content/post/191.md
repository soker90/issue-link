## Qué es

Million React es una plantilla starter de código abierto creada por Aiden Bai, autor de Million.js. Es un esqueleto mínimo de Vite + React con Million.js ya enganchado, pensado para probar sobre la marcha las optimizaciones de renderizado de la librería sin configurar nada. El repositorio está archivado por su autor desde septiembre de 2023, por lo que es de solo lectura. Se distribuye bajo licencia MIT.

## Para qué sirve

- Arrancar un proyecto React + Vite con Million.js ya configurado, sin partir de cero.
- Probar el "block virtual DOM" de Million.js sobre componentes reales.
- Servir de referencia mínima de cómo integrar Million.js en una app de Vite.
- Evitar la configuración inicial repetitiva al evaluar la librería.

## Cuándo usarlo

Solo como punto de partida rápido para experimentar con Million.js en un entorno Vite. Al estar archivado, conviene tratarlo como ejemplo histórico: para un proyecto real es mejor crear un proyecto Vite actual e instalar Million.js (`aidenybai/million`) siguiendo su guía oficial, que cubre integraciones mantenidas para Vite, Next.js y Astro.

## Ejemplo

Clonar el starter y levantar el entorno de desarrollo:

```bash
git clone https://github.com/aidenybai/million-react.git
cd million-react
npm install
npm run dev
```

Sobre un proyecto Vite propio, la forma actual de añadir Million.js es mediante su plugin de compilación en vite.config.js:

```js
import million from "million/compiler";
import react from "@vitejs/plugin-react";
import { defineConfig } from "vite";

export default defineConfig({
  plugins: [million.vite({ auto: true }), react()],
});
```

## Puntos clave

- Plantilla con la estructura básica de Vite + React y Million.js ya integrado.
- Million.js es un compilador optimizador que actualiza directamente los nodos del DOM afectados en lugar de rehacer la reconciliación completa de React.
- Mantenida por el mismo autor que la librería Million.js.
- Código abierto bajo licencia MIT.

## Ten en cuenta

El repositorio está archivado (solo lectura) desde 2023, así que sus dependencias y la versión de Million.js que incluye están desfasadas: úsalo como referencia, no como base de un proyecto nuevo. La librería Million.js (`aidenybai/million`) sigue siendo la referencia y se instala aparte con su versión y plugin actuales. Ten en cuenta además que su autor ha reorientado su actividad hacia otros proyectos (React Scan y herramientas de IA), por lo que conviene verificar el estado de mantenimiento de Million.js antes de adoptarlo en algo nuevo.
