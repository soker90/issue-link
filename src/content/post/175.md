## Qué es

Electron Vite React es un template o starter open source. electron: Electron + Vite + React + Sass boilerplate.

## Para qué sirve

- Arrancar un proyecto rápido con la estructura de Electron Vite React.
- Aprender patrones de un proyecto real bien configurado.
- Evitar la configuración inicial repetitiva en cada proyecto.
- Integrar en proyectos React existentes.

## Cuándo usarlo

- Cuando arrancas un proyecto nuevo y quieres saltar la configuración inicial con una base bien estructurada.

## Puntos clave

- Estructura de proyecto lista para producción.
- Configuración inicial ya resuelta.
- Open source y disponible en GitHub.
- Diseñado para el ecosistema React.

## Ten en cuenta

- Personaliza el template a fondo para que tu proyecto tenga identidad propia.
- Consulta la documentación oficial para obtener la información más actualizada.
