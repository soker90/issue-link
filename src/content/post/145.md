## Qué es

Astro Art Portfolio es una plantilla open source creada por Ema Suriano para construir el portfolio de un artista. Está hecha con Astro y TypeScript, usa Tailwind CSS para el diseño y la API de Notion como gestor de contenido, de modo que las obras se administran desde una base de datos de Notion sin tocar el código. La galería de imágenes se apoya en PhotoSwipe.

## Para qué sirve

- Publicar un portfolio de obras con una galería de imágenes navegable y con lightbox.
- Gestionar el catálogo (títulos, imágenes, datos) desde Notion en lugar de editar archivos.
- Tener una web rápida y estática generada con Astro, lista para desplegar en Netlify.
- Personalizar colores, tema claro/oscuro y secciones partiendo de una base ya montada.

## Cuándo usarlo

Es una buena opción si eres artista o ilustrador y quieres una web sencilla, rápida y centrada en mostrar imágenes, gestionando el contenido desde Notion sin escribir código. Si prefieres un CMS más completo o no usas Notion, conviene valorar plantillas de Astro basadas en colecciones de contenido locales o en otros CMS.

## Ejemplo

Flujo básico para ponerla en marcha:

1. Duplica la plantilla de Notion enlazada en el repositorio y sustituye las imágenes y textos por los tuyos.
2. Clona el repositorio e instala las dependencias.
3. Crea un archivo .env con las credenciales de la API de Notion (token de integración e ID de la base de datos).
4. Arranca el proyecto en local con Astro y, cuando esté listo, despliégalo en Netlify.

Comandos habituales:

git clone https://github.com/EmaSuriano/astro-art-portfolio
cd astro-art-portfolio
yarn install
yarn dev

## Puntos clave

- Contenido gestionado desde Notion mediante su API, sin editar el código para añadir obras.
- Galería con lightbox basada en PhotoSwipe e imágenes optimizadas (WebP).
- Soporte de tema claro/oscuro y estilos con Tailwind CSS.
- Demo pública desplegada en Netlify y licencia MIT.

## Ten en cuenta

Depende de la API de Notion, por lo que necesitas una cuenta de Notion y configurar un token de integración; si Notion cambia su API o limita el acceso, puede afectar al funcionamiento. Es una plantilla de un proyecto personal (no un producto con soporte comercial), así que está pensada para adaptarse y mantenerse por cuenta propia. El proyecto está activo y publicado bajo licencia MIT.
