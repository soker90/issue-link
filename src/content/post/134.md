## Qué es

Tabler Icons es una colección open source de iconos SVG creada por el equipo de Tabler. Reúne más de 6.100 iconos (variantes outline y filled) diseñados sobre una rejilla de 24x24 píxeles con un trazo de 2px, lo que les da un aspecto coherente. Todo el conjunto se publica bajo licencia MIT y se distribuye tanto como archivos SVG sueltos como en paquetes específicos para distintos frameworks.

## Para qué sirve

- Incorporar iconos uniformes a una interfaz web o app sin pagar licencias.
- Usar componentes de icono nativos en React, Vue, Svelte o Angular.
- Cargar iconos vía webfont o sprite SVG cuando no se trabaja con un framework.
- Personalizar tamaño, color y grosor de trazo al ser SVG editables.

## Cuándo usarlo

Es buena opción cuando necesitas un set amplio, gratuito y con estilo homogéneo para tu producto. Encaja especialmente bien con proyectos que ya usan el ecosistema Tabler (dashboard, plantillas), aunque funciona en cualquier stack. Como alternativas están Lucide, Heroicons o Phosphor; Tabler destaca por el número de iconos y su diseño consistente de trazo.

## Ejemplo

Instalación y uso del paquete para React:

npm install @tabler/icons-react

import { IconHeart } from '@tabler/icons-react';

function App() {
return <IconHeart size={24} color="red" stroke={2} />;
}

Para proyectos sin framework, basta con incluir el SVG directamente o usar la webfont desde un CDN.

## Puntos clave

- Más de 6.100 iconos (5.093 outline y 1.053 filled) en constante crecimiento.
- Paquetes oficiales para React, Vue, Svelte (4 y 5) y Angular, además de SVG, sprite y webfont.
- Licencia MIT que permite uso personal y comercial sin restricciones.

## Ten en cuenta

El proyecto está activo y muy mantenido. La versión open source incluye solo el código fuente SVG; si necesitas formatos compilados listos para usar (PNG, PDF, Figma, Sketch) o el bundle completo con plantillas, hay opciones de pago de un solo abono, pero no son necesarias para usar los iconos en web.
