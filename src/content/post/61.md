## Qué es

Bolt.new es una plataforma de desarrollo web asistida por IA creada por StackBlitz. Permite generar, ejecutar, editar y desplegar aplicaciones full-stack directamente en el navegador, sin instalar nada en local, gracias a la tecnología WebContainers (un entorno Node.js que corre dentro del propio navegador). Su código base es open source con licencia MIT y está escrito principalmente en TypeScript sobre Remix y Vite. Existen dos caras: el producto SaaS en bolt.new y el repositorio open source que sirve de plantilla para construir tu propio agente.

## Para qué sirve

- Crear una aplicación o sitio web completos a partir de un prompt en lenguaje natural.
- Prototipar ideas rápidamente con frontend y backend en un mismo entorno.
- Editar el código generado en vivo y ver el resultado al instante en la vista previa.
- Desplegar el proyecto resultante sin configurar infraestructura manualmente.

## Cuándo usarlo

Es útil cuando quieres pasar de una idea a una app funcional en minutos sin montar un entorno local, especialmente para prototipos, MVP o pruebas de concepto. Si lo que buscas es la versión open source con libertad para elegir el modelo de IA (OpenAI, Anthropic, Ollama, Gemini, etc.), conviene mirar bolt.diy, el fork comunitario oficial mantenido en la organización stackblitz-labs.

## Ejemplo

Flujo típico de uso en la plataforma:

```
1. Abrir bolt.new e iniciar sesión.
2. Escribir un prompt describiendo la app:
   "Crea una app de lista de tareas con React y persistencia en localStorage".
3. Bolt genera la estructura del proyecto, instala dependencias
   y arranca un servidor de desarrollo en el WebContainer.
4. Ajustar el resultado con prompts adicionales o editando el código
   directamente en el editor integrado.
5. Desplegar el proyecto desde la propia interfaz.
```

Para construir tu propio agente partiendo del código open source, puedes clonar el repositorio y arrancarlo en local:

```
git clone https://github.com/stackblitz/bolt.new
cd bolt.new
pnpm install
pnpm run dev
```

## Puntos clave

- Entorno full-stack que se ejecuta íntegramente en el navegador mediante WebContainers, sin servidores remotos para el desarrollo.
- Código base open source (MIT) en TypeScript sobre Remix y Vite, pensado como plantilla para crear agentes de IA propios.
- Soporta la mayoría de frameworks y librerías populares de JavaScript.
- El producto comercial añade enrutado automático de modelos, hosting, bases de datos, autenticación y analítica.

## Ten en cuenta

El producto bolt.new es freemium: hay funcionalidades gratuitas, pero las generaciones avanzadas y los planes de pago consumen créditos. La plataforma estaba en beta y StackBlitz la sigue mejorando, así que el comportamiento puede cambiar. No confundas el repositorio open source (una plantilla base) con el SaaS completo: el repo no incluye toda la funcionalidad del producto. Si necesitas elegir libremente el modelo de IA o autoalojar la herramienta, bolt.diy es la alternativa más adecuada.
