## Qué es

OpenStatus es una plataforma open source para monitorizar endpoints HTTP y publicar una pagina de estado. Sirve para saber si una web, API o servicio esta funcionando correctamente y para comunicar incidencias de forma transparente.

## Para qué sirve

- Comprobar periodicamente si una URL responde.
- Detectar caidas, latencia alta o errores HTTP.
- Publicar una status page para usuarios, clientes o equipo interno.
- Tener una alternativa abierta a servicios comerciales de uptime.

## Cuándo usarlo

Usalo si tienes APIs, landing pages, dashboards o aplicaciones publicas y quieres una forma sencilla de saber si estan disponibles. Es especialmente interesante si quieres controlar mas la infraestructura o evitar depender completamente de un SaaS cerrado.

## Puntos clave

- Enfoque open source.
- Pensado para monitorizacion y comunicacion de estado.
- Encaja bien en proyectos personales, productos pequenos y equipos que quieren visibilidad basica de disponibilidad.

## Ten en cuenta

Si solo necesitas una solucion basada en GitHub Actions y GitHub Pages, Upptime puede ser mas simple. Si necesitas alertas avanzadas, checks distribuidos o SLA complejos, conviene compararlo con herramientas dedicadas.
