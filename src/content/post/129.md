## Qué es

Tailcast es una plantilla de sitio web open source creada por Matt (matt765), pensada para la landing de una startup ficticia. Está construida con Astro 6, Tailwind CSS 4 y TypeScript, con un diseño de tema oscuro, responsive y listo para personalizar. Se distribuye bajo licencia MIT.

## Para qué sirve

- Lanzar rápido la web de un producto o SaaS sin partir de cero.
- Aprovechar un diseño de landing moderno ya maquetado y responsive.
- Aprender una estructura de proyecto Astro bien organizada.
- Reutilizar páginas y secciones (hero, precios, FAQ, blog) como base.

## Cuándo usarlo

Encaja cuando necesitas una landing presentable en poco tiempo y trabajas con el stack de Astro y Tailwind. Es ideal para presentar un producto, una startup o un servicio. Si buscas un sitio con contenido editable por terceros o un CMS, tendrás que integrarlo aparte; la plantilla cubre la parte de presentación, no la gestión de contenido.

## Ejemplo

Clonar el repositorio y arrancar el entorno de desarrollo:

git clone https://github.com/matt765/Tailcast.git
cd Tailcast
npm install
npm run dev

El servidor de desarrollo queda disponible en localhost y desde ahí se editan las páginas de la plantilla (home, about, services, blog, careers, contact y 404).

## Puntos clave

- 8 páginas prediseñadas: home, about, services, blog, careers, contact, artículo y 404.
- Optimización SEO con sitemap y etiquetas Open Graph.
- Accesibilidad con HTML semántico y atributos ARIA.
- Transiciones de página suaves mediante el Client Router de Astro.
- Diseño mobile-first y pipeline de CI ya configurado con ESLint y Prettier.

## Ten en cuenta

Es una plantilla de presentación, no un framework ni un producto con backend: el contenido va en el propio código y no incluye CMS ni panel de administración. Conviene personalizar a fondo textos, colores e imágenes para que tu web tenga identidad propia y no se reconozca como la demo original.
