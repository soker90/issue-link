## Qué es

Astrofy es una plantilla (template) gratuita y de código abierto creada por Manuel Ernesto García para construir un sitio web de portfolio personal. Está hecha con Astro, TailwindCSS y DaisyUI, y se distribuye bajo licencia MIT. Trae secciones ya montadas (blog, CV, proyectos, servicios, tienda y contacto) para tener un sitio en marcha sin partir de cero.

## Para qué sirve

- Publicar un portfolio personal con páginas de proyectos, servicios y CV ya estructuradas.
- Mantener un blog con paginación y feed RSS sin configurarlo desde cero.
- Añadir una sección de tienda para vender productos o servicios.
- Cambiar la apariencia rápidamente gracias a los más de 30 temas de DaisyUI.

## Cuándo usarlo

- Cuando quieres una web personal o de freelance en Astro sin montar la estructura y el estilo a mano.
- Cuando prefieres una base open source y editable como punto de partida para personalizar.
- Si solo necesitas un blog o una landing muy simple, una plantilla con tantas secciones puede sobrar; valora alternativas más ligeras.

## Ejemplo

Clonar la plantilla y arrancar el entorno de desarrollo:

git clone https://github.com/manuelernestog/astrofy.git
cd astrofy
npm install
npm run dev

Las entradas del blog y los proyectos se gestionan como colecciones de contenido de Astro, añadiendo archivos Markdown en las carpetas de contenido. Para generar el sitio estático listo para publicar se ejecuta npm run build.

## Puntos clave

- Secciones predefinidas: blog, CV, proyectos, servicios, tienda y contacto.
- Generación de feed RSS integrada para el blog.
- Más de 30 temas mediante DaisyUI, con soporte de modo claro y oscuro.
- Diseño responsive con navegación lateral.
- Contenido gestionado con colecciones de contenido de Astro y Markdown.

## Ten en cuenta

- Es un punto de partida: conviene personalizar contenidos, estilos y secciones para que el sitio tenga identidad propia.
- Incluye más secciones de las que muchos portfolios necesitan; puedes eliminar las que no uses.
- Requiere conocimientos básicos de Astro y TailwindCSS para sacarle partido y desplegarla (por ejemplo en Netlify o Vercel).
