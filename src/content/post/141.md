## Qué es

vite-project es un proyecto de ejemplo creado por Cory House (autor de cursos de React) que muestra cómo montar una aplicación React moderna con Vite como bundler y Vitest como framework de testing. Está escrito mayormente en TypeScript y trae preconfigurados ESLint (con la configuración de react-app vía vite-plugin-eslint), Tailwind CSS, Storybook y Prettier. No es una librería ni una herramienta publicada, sino un repositorio de referencia/demo asociado a material formativo.

## Para qué sirve

- Ver cómo se conecta Vitest con Testing Library en un proyecto React real.
- Partir de una base ya configurada con ESLint, Tailwind, Storybook y Prettier.
- Consultar la configuración concreta de Vite para React y testing.
- Revisar la rama `library` como ejemplo de cómo usar Vite para generar una librería.

## Cuándo usarlo

Útil cuando quieres una referencia sencilla de cómo encaja todo el tooling (Vite + Vitest + Testing Library + Tailwind + Storybook) en una app React, o como punto de partida para experimentar. Para arrancar un proyecto nuevo de producción, hoy es más recomendable `npm create vite@latest` con la plantilla react-ts, que está mantenida y actualizada; este repositorio sirve sobre todo como apoyo didáctico.

## Ejemplo

Clonar el repositorio y ejecutarlo en local:

    git clone https://github.com/coryhouse/vite-project.git
    cd vite-project
    npm install
    npm run dev      # arranca el servidor de desarrollo de Vite
    npm test         # ejecuta los tests con Vitest

## Puntos clave

- Integra Vitest con Testing Library para tests de componentes React.
- Trae ESLint, Tailwind CSS, Storybook y Prettier ya enchufados.
- Incluye una rama `library` con un ejemplo de Vite generando una librería.

## Ten en cuenta

El repositorio no declara licencia, por lo que por defecto no concede permisos de uso; si quieres reutilizar su código, conviene preguntar al autor. Además, su último commit es de noviembre de 2022: las versiones de Vite, Vitest y el resto de dependencias están desfasadas, así que úsalo como referencia conceptual más que como base actualizada. Para empezar un proyecto nuevo, mejor el scaffolding oficial de Vite.
